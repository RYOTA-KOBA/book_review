# 環境構築

## セットアップコマンド
```sh
$ bin/setup
```

## 起動
```sh
$ docker-compose up
```

## コンテナへ接続
```sh
$ docker-compose run --rm frontend bash
$ docker-compose run --rm backend bash
```

## データベース
| 項目 | 値 |
| --- | --- |
| Host | 127.0.0.1 |
| Username | root |
| Password | root |
| Database | app_development |
